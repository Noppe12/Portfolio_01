# クリエイター画像投稿サイト

## サイト概要
個人クリエイターが自分で簡単に管理できるサイト

### サイトテーマ
クリエイター用の登録制サイト

### テーマを選んだ理由
近年個人でSNSを活用したクリエイターが増えているので、youtubeのメンバー機能のようなサイトがあれば
より一層個人クリエイターはやりやすくなるのではないのかなと思い、個人サイトを作れるサイトを考えました。

### ターゲットユーザ
ツイッターやyoutube、pixivなどで個人で投稿しているクリエイター向け
